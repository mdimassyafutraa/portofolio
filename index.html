Portofolio

Buku Tamu UPT Balai Yasa Lahat

Situs web yang saya bangun adalah platform pencatatan tamu yang menawarkan kemudahan dalam mencatat kedatangan tamu ke UPT Balai Yasa Lahat. Dengan fitur-fitur seperti input data tamu, pembuatan QR code, pemindaian QR code, pencatatan tamu dalam file Excel, serta visualisasi jumlah tamu per hari, minggu, dan bulan, website ini memberikan solusi lengkap untuk manajemen kedatangan tamu yang efisien dan terorganisir.

HTML, CSS Tailwind, Laravel, Javascript.

Website Penyewaan Gedung

Situs ini merupakan platform penyewaan gedung seni yang menyediakan kemudahan dalam pemesanan. Dengan fitur input data peminjam serta kalender jadwal, pengguna dapat dengan mudah memasukkan informasi pemesanan mereka dan melihat ketersediaan jadwal sewa gedung secara real-time. Dengan demikian, situs ini menjadi sarana praktis bagi peminjam dalam merencanakan acara mereka dengan tepat waktu.

HTML, CSS Tailwind, Laravel, Javascript.
